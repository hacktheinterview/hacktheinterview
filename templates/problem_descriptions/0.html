You're given a singly linked list 1 -> 2 -> 3.

You have to reverse the linked list into 3 -> 2 -> 1 .


(i.e., 0 1 2 4 5 6 7 might become 4 5 6 7 0 1 2).


You are given a target value to search. If found in the array, return its index, otherwise return -1.

You may assume no duplicate exists in the array.

For example if the input array is [4 5 6 7 0 1 2] and target = 4, the output will be 0 because 4 is found in position 0.
