You're given a two numbers represented as linked lists

For example, number 123 is represented as 1 -> 2 -> 3 and number 56789 is represented as 5 -> 6 -> 7 -> 8 -> 9

Given two numbers represented as linked lists, return a single list which represents the addition of two linked lists.

Examples: Input: List1: 1 -> 2 -> 3 -> 4 List2: 1 -> 2 -> 3 -> 4 Output: 2 -> 4 -> 6 -> 8 Input: List 1: 9 -> 9 -> 9 List 2: 1 -> 1 Output: 1 -> 0 -> 0 -> 0
